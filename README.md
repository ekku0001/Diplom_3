# Diplom part 3

## Стек технологий

- Java 11
- Apache Maven 3.8.1
- Junit 4.13.2
- RestAssured 5.3.0
- Selenium 4.8.0
- Allure 2.21.0

## Команды

- mvn clean: очищает проект и удаляет все файлы, созданные предыдущей сборкой.
- mvn test: запускает тесты для проекта.
- mvn allure:serve: генерирует отчет и открывает его.
